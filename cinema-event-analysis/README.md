# 🎬 Cinema Event Analysis

## 📌 Project Overview

This project analyzes user interaction events in an online cinema player. The objective is to identify the most active users, examine their behavior, detect inconsistencies in event sequences, and explore usage patterns across devices and browsers.

The analysis demonstrates data filtering, aggregation, validation, and visualization using Python and the Pandas library.

---

## 📂 Dataset

The dataset contains user event logs with the following fields:

- `account_id` — user identifier
- `server_time` — event timestamp
- `screen_type` — page type
- `action_id` — event type
- `device_type` — device category
- `user_browser` — browser

Only events related to the **player** screen were included in the analysis.

---

## 🎯 Objectives

- Filter player-related events
- Identify the 10 most active users
- Analyze browser usage among the most active users
- Calculate the distribution of event types
- Detect inconsistent user behavior in event sequences
- Explore anomalies by device type and browser
- Visualize key findings
